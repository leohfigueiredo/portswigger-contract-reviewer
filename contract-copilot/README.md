# Contract Copilot

An interactive AI-assisted contract review and triage application. Designed to automate the review of low-stakes agreements (NDAs, Order Forms) and flag critical compliance and geopolitical risks.

## 🚀 Getting Started

1. Navigate to the application folder:
   ```bash
   cd contract-copilot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the local dev server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## 💡 Tech Stack
* Core: React, Vite, Lucide React
* Styling: Custom HSL glassmorphism dark-mode CSS
* AI: Google Gemini SDK integration, custom client-side heuristics engine, OpenAI/Claude/Ollama endpoint router.
